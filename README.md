### Custom ESP32 Bluetooth speaker
Creating a custom bluetooth speaker using esp32.Esp32 for bluetooth connectivity , a DAC (Digital-to-Analog Converter) for audio quality, potentially an amplifier, and a battery with a charging circuit.
